# Boot OS from Virtual Hard Drive (VHD)

Download the Sharepint VHD from http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=27417 ->2010-10a and merge them.

Load the VHD using Virtual Box.

  • Shrink the VHD to 78GB from 130GB (Don't worry about this if you have 130GB space to be used by Sharepoint)

Download the VHD Resizer from http://vmtoolkit.com/files/folders/converters/entry87.aspx

Make the VHD of 80GB with fixed size (no dynamic)

Add to the Boot menu:

  1. bcdedit /copy {current} /d "Sharepoint"
  2. Use Edit >> Mark and Edit >> Copy to copy the GUID so it’s easy to insert into the remaining commands.
  3. bcdedit /set {guid} device vhd=[d:]\2010a.vhd
  4. bcdedit /set {guid} osdevice vhd=[d:]\2010a.vhd
  5. bcdedit /set {guid} detecthal on

To Remove from Boot menu: bcdedit /delete {guid} /cleanup

After boot: Enable the Wifi Service on windows 2008 Server (by defaule, it's disabled) -> Feature Manager -> Add Feature, check the Wifi lan service
